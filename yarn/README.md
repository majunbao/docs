# YARN
前端代码的包管理器。

## 常用命令

- yarn init                初始化新项目
- yarn add                 添加依赖包
- yarn add --dev           添加到devDependencies类别
- yarn add --peer          添加到peerDependencies类别
- yarn add --optional      添加到optionalDependencies类别
- yarn upgrade             升级依赖
- yarn remove              移除依赖
- yarn OR yarn install     安装项目的全部依赖
